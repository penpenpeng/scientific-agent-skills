# 1. 基本信息提取要求

## 必须提取的字段

- **标题**：论文完整标题
- **作者**：所有作者姓名
- **单位**：所有作者单位，保持层级关系
- **期刊/年份**：期刊名称和发表年份
- **DOI**：完整的DOI链接
- **收稿/修回/接受日期**：Received/Revised/Accepted日期
- **关键词**：论文关键词列表

## 提取规范

1. 作者单位按照原文层级记录，多个单位用编号标注
2. 日期格式统一为：YYYY年MM月DD日
3. 关键词之间用顿号（、）分隔
4. 通讯作者和邮箱需单独标注

## 输出格式

```markdown
## 1. 基本信息
- **作者**：XXX, XXX, XXX
- **单位**：
  - 1. 单位一名称
  - 2. 单位二名称
- **期刊/年份**：期刊名 / 年份
- **DOI**：https://doi.org/xxx
- **收稿/修回/接受日期**：收稿日期 / 修回日期 / 接受日期
- **关键词**：关键词1、关键词2、关键词3
```

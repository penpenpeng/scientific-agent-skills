# 3. 核心结果提取要求

## 必须遵守的规则

1. **所有结论性语句必须标注原文出处**
   - 格式：`（原文位置：结果部分 X 节 / 图 X / 表 X）`
   - 如有多个出处，用逗号分隔

2. **按主题组织结果**
   - 每个小节对应一个主要发现
   - 使用清晰的标题

3. **与实验设计交叉验证**
   - 确保结果中提到的所有指标都在实验设计中记录
   - 如有不一致，以结果为准并标注

## 提取内容

- **主要发现**：每项研究的核心结果
- **数据支持**：关键数据、统计结果
- **对照关系**：与对照组的比较
- **机制通路**：涉及的信号通路、分子机制

## 输出格式

```markdown
## 3. 核心结果

### 3.1 [主题1]（原文位置：Results X.X / 图X）
- **主要发现**：...（原文位置：...）
- **数据支持**：...（原文位置：...）

### 3.2 [主题2]（原文位置：Results X.X）
- **发现1**：...（原文位置：...）
- **发现2**：...（原文位置：...）
```

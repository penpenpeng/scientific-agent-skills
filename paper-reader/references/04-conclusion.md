# 4. 结论与创新点提取要求

## 4.1 主要结论

### 提取规范
1. 从Abstract和Discussion中提取核心结论
2. 每个结论需标注原文出处
3. 结论应简洁明了，突出研究意义

### 输出格式
```markdown
### 主要结论
1. **结论1**（原文位置：Abstract / Discussion）
2. **结论2**（原文位置：Discussion）
...
```

---

## 4.2 创新点

### 提取规范
1. 识别研究的创新性贡献
2. 与已有研究对比，突出新颖性
3. 包括方法创新、发现创新、应用创新等

### 输出格式
```markdown
### 创新点
1. **创新点1**：...
2. **创新点2**：...
```

---

## 4.3 机制总结（如有）

### 提取规范
1. 使用箭头图表示信号通路或调控关系
2. 标注上下游分子
3. 展示干预策略

### 输出格式
```markdown
### 机制总结
```
[机制流程图]
```
```

---

## 完整输出格式

```markdown
## 4. 结论与创新点

### 主要结论
1. **...**（原文位置：...）

### 创新点
1. **...**

### 机制总结（如有）
```
[流程图]
```
```
